== Description ==

Site Keywords allows you to keep a list of keywords and assign them to links (internal and external).
When a keyword is entered and submitted the site will go directly to the attached URL, if there is no keyword, the default WP search is used.


== Installation ==

1: Upload the "Site Keyword" plugin to your blog.

2: Activate it.

3: Add keywords.
